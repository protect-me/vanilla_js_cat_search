프로젝트 실행 : `$ npm run dev`

## 환경설정
`$ npm init -y` : npm 초기화
`$ npm install parcel-bundler -D` : parcel-bundler를 개발용(-D)으로 설치
`.gitignore` : git에서 관리하지 않을 파일 리스트 추가
